# 🤖 AGENT INSTRUCTIONS: ROOT

Välkommen till Local Ledger.

1. **Arkitektur:** Läs ALLTID `ARCHITECTURE.md` i denna mapp innan du påbörjar kodning för att förstå beroenden och strukturen. Följ "Check-Update-Act"-principen.
2. **PRD:** De legala och funktionella kraven för projektet finns i `docs/`. Avvik aldrig från `Master_PRD.md` eller `Compliance_Spec.md`.
3. **Mappspecifika Regler:** Gå in i respektive mapp (t.ex. `internal/` eller `frontend/`) och läs `_AGENT_INSTRUCTIONS.md` där för specifika kodkonventioner innan du modifierar dem.
4. **Kodbas:** Backend är skriven i Go, Frontend i HTML/JS/CSS utan tunga ramverk. Databasen är SQLite.
